# CI Web Project

Proyek sederhana untuk menampilkan identitas mahasiswa dan mengimplementasikan Continuous Integration (CI) menggunakan GitHub Actions.

## Fitur
- Web statis (`index.html`)
- Pengujian otomatis (`test_script.py`)
- Pipeline CI otomatis di GitHub Actions

## Cara kerja
Setiap kali ada push ke branch `main`, GitHub Actions akan menjalankan pengujian otomatis dan menampilkan hasilnya di tab **Actions**.
